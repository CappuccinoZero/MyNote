回调函数的 Kotlin的lambda表达式简化

普通的回调：
View.setListener(object : touchListener{
    fun onSuccess(msg:String){
        //to do
    }
})
三种基本lambda表达式：
View.setListener({
    msg:String ->
    //to do
})
View.setListener({
    msg->
    //to do
})
如果不需要使用参数
View.setListener({
    //to do
})

如果回调参数在最后一个
View.setListener(){
    //to do
}
  
只有一个参数的话
此时it隐指参数
View.setListener{
    //to do
}
