一只青蛙一次可以跳上1级台阶，也可以跳上2级……它也可以跳上n级。求该青蛙跳上一个n级的台阶总共有多少种跳法。

根据题意可以找到规律：
F(n) =F(0)+ F(1)+F(2)+F(3)+…+F(n-1)
 
public int JumpFloorII(int target) {
        List<Integer> arrays = new ArrayList<>();
        arrays.add(1);//F.0
        arrays.add(1);//F.1
        for (int i = 2 ;i<=target ; i++){
            int sum = 0;
            for(int j=0;j<arrays.size();j++){
                sum += arrays.get(j);
            }
            arrays.add(sum);
        }
        return arrays.get(target);
    }

根据：
F(n) =F(0)+ F(1)+F(2)+F(3)+…+F(n-1)
F(n+1) =F(0)+ F(1)+F(2)+F(3)+…+F(n-1)+F(n)
可得 F(n+1) = 2F(n)

private int JumpFloorII(int target) {
    int sum = 1;//f[1]
    for(int i=2;i<=target;i++){
        sum *= 2;
    }
    return sum;
}
