我们可以用2*1的小矩形横着或者竖着去覆盖更大的矩形。请问用n个2*1的小矩形无重叠地覆盖一个2*n的大矩形，总共有多少种方法？

public int RectCover(int target) {
        if(target<=1)
            return target;
        if(target==2)
            return 2;
        return RectCover(target-1) + RectCover(target-2);
    }

解题思想，斐波那契算法

当n等于0时，为0种方法
当n等于1时，只有一种方法
当n等于2时，有横着放两个，竖着放两个，共两种放法
当n等于3时，这时只需要考虑最后一步是怎么放的，最后一步只有两种放法，横着放，竖着放
    1.如果横着放，放数就等于n=n-1时候的方数
    2.如果竖着放，则必然是连个矩形同时竖着放，所以方数就等于n=n-2时候放数
最后将两种放数加上就是n时的放法。
